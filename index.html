Open Question 2 · Python Code
Write a complete Python program that performs the following steps:
  1. Ask the user to enter three integer numbers: a, b, and c.
  2. Compute and print the average of the three numbers as a float.
  3. Determine and print whether the integer part of the average is even or odd.
  4. Print the average rounded to 2 decimal places using round().
⚠ Your program must produce clearly labelled output for each step.
Hints:
  • Use int(input(...)) to read integers from the user.
  • Use / (not //) so the average is a float.
  • Use int(average) % 2 to check even or odd.
  • Use round(value, 2) for 2-decimal rounding.
